# 6. 安装less 支持

```bash
npm i less less-loader --save
```

注意一点： 在模版里面添加 lang='less'

```less
<style  lang='less' scoped>
#app {
  .nav {
    .nav-item {
      color: red;
    }
  }
}
</style>
```
